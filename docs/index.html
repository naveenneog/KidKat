Built the right way

Within YouTube's Terms — by design

The “log into YouTube Kids and override the algorithm” idea isn’t allowed or even possible. KidKat achieves the same goal the compliant way.

❌
No YouTube Kids login
There is no public Kids API — child accounts are COPPA-restricted.
✅
Official Data API v3 discovery
Search + metadata only, with strict Safe Search.
❌
No algorithm override
We never read or alter YouTube’s recommendations.
✅
Official IFrame player
No stream extraction — creators keep their views.
❌
No child PII
Profiles & controls live on-device. No tracking SDKs.
✅
Parent-curated allowlist
Our own curation: trusted channels + Education/Science only.
What's inside

Features kids love, parents trust

Everything is tuned to keep learning fun and finite.

🧩

Finite sessions

A set number of videos, then a friendly break screen. No infinite feed.

⏱️

Daily time limit

When time’s up, the app locks until tomorrow. Parents can extend.

📺

Full-screen shorts

Immersive full-screen playback. Advancing is a deliberate tap, never a swipe; related videos stay on-channel.

🔒

Parent PIN gate

All settings are protected behind a 4-digit parent PIN.

🧒

Age-appropriate channels

Set your child's age (3–5, 6–8, 9–12) and KidKat pre-loads trusted channels and tailors searches to fit.

🎓

12 learning topics

Science, Space, Animals, Math, Reading, Art, Coding, and more.

🛡️

Strict Safe Search

Embeddable-only, short-only (≤1 or ≤4 min), category-filtered.

📱

Android & iOS

One Flutter codebase. Riverpod, go_router, official IFrame player.
